# Data Conversion Toolkit

各種データ形式（数値、文字列、エンコード、フォーマットなど）を相互に変換・解析するための開発者向けツール集です。

Vite + React で構築されており、ビルド後は **単一の HTML ファイル (`dist/index.html`)** として出力されます。サーバーを必要とせず、ブラウザで直接開くだけでローカル環境でもポータブルに利用できるのが特徴です。

## 🚀 特徴
- **ポータブル**: ビルド結果は1つの HTML ファイルに集約（JS/CSS をインライン化）。
- **高速動作**: React による SPA 構成で、タブ切り替えや変換処理がスムーズ。
- **自動変換**: 入力項目を変更すると、即座に他の形式へ同期更新されます。
- **モダンな UI**: Sidebar とタブによる整理されたカテゴリ分け。

## 💻 開発環境のセットアップ

### コマンド
```bash
# 依存関係のインストール
npm install

# 開発用サーバーの起動
npm run dev

# 単一 HTML ファイルのビルド
npm run build
```

ビルド後、`dist/index.html` をブラウザで開くことで、すべての機能がオフラインで利用可能になります。
